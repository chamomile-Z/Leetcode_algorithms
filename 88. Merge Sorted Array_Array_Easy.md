### My Solution (**Merge and Sort**)
```Python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        """
        Do not return anything, modify nums1 in-place instead.
        """
        if len(nums2) == 0:
            nums1 = nums1
        else:
            for i in range(m, m+n):
                nums1[i] = nums2[i-m]
            nums1 = nums1.sort()
```

It equals to:

```Python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        """
        Do not return anything, modify nums1 in-place instead.
        """
        nums1[:] = sorted(nums1[:m] + nums2)
```

```diff
- Time Complexity: O((m+n)log(m+n))
- Space Complexity: O(1)
```

```Python
class Solution:
  def merge(self, nums1: List[int], m:int, nums2: List[int], n:int) -> None:
    """
    Do not return anything, modify nums1 in-place instead
    """
    # p1 is the pointer to the end element of nums1, p is the pointer to the end place of nums1
    # p2 is the pointer to the end element of nums2
    p1 = m - 1
    p2 = n - 1
    p = m + n - 1
    
    while p1 >= 0 and p2 >= 0:
      if nums[p1] < nums[p2]:
        nums[p] = nums[p2]
        p2 -= 1
      else:
        nums[p] = nums[p1]
        p1 -= 1
      p -= 1
      
    # adding missing elements from num2
    nums1[:p2+1] = nums2[:p2+1]
```

```diff
- Time Complexity: (m+n)log(m+n)
- Space Complexity: O(1)
```
