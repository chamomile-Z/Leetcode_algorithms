# My Solution (Hash Table)
```Python
def twoSum(self, numbers: List[int], target: int) -> List[int]:
  dic = {}
  
  for ind, num in enumerate(numbers):
    if num not in dic:
      dic[target - num] = ind + 1
     else:
      return [dic[num], ind+1]
```

## Method
- Time Complexity: O(n)
- Space Complexity: O(n)
