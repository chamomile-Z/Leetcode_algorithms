# My Solution (set function)
```Python
    def findDisappearedNumbers(self, nums: List[int]) -> List[int]:
        n = len(nums)
        uniquenums = set(nums)
        res = []
        
        for num in range(1, n+1):
            if num not in uniquenums:
                res.append(num)
                
        return res
```

## Method
- Time Complexity: O(n)
- Space Complexity: O(n)
