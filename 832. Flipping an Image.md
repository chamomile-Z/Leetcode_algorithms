# My Solution (reverse function for list)
```Python
    def flipAndInvertImage(self, A: List[List[int]]) -> List[List[int]]:
        # flipping
        for a in A:
            a.reverse()
            # inverting
            for j in range(len(a)):
                if a[j] == 1: 
                    a[j] = 0
                else:
                    a[j] = 1
        return A
```

## Method
- Time Complexity: O(n * m)
- Space Complexity: O(1)
